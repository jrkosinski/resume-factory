## Context

Pairtrade Finder is a pairs trading analysis tool produced by Event Driven Investor. Pairs trading is a market-neutral quantitative trading strategy that identifies two financial assets whose prices historically move together (are correlated), then trades the divergence when their price relationship temporarily breaks down. The accuracy of the correlation calculation is central to the strategy's validity — a flawed correlation metric produces wrong signals and bad trades.

The engagement was focused on correctness and code quality: fixing broken statistical math, improving the underlying algorithms, modernizing the codebase structure, and adding test coverage to make the codebase more reliable and maintainable.

## Technical Work

### Pearson's Coefficient Correction

The existing implementation of Pearson's correlation coefficient — the core statistical measure used to assess whether two asset price series move together — contained calculation errors. Identified and corrected these errors to ensure the correlation output was mathematically accurate, which is a prerequisite for the pairs trading signals to be trustworthy.

### Correlation Algorithm Improvement

Beyond fixing the immediate calculation errors, improved the broader algorithms used to analyze correlation between distinct financial asset price series. This covered how price series data was prepared, normalized, and fed into the correlation analysis.

### Codebase Refactoring

Refactored the C# Windows desktop application code to improve organization, extensibility, and modularity — making the codebase easier to maintain, extend, and reason about. This was structural improvement rather than feature addition.

### Automated Testing and Bug Discovery

Added automated tests to the existing codebase. The tests served a dual purpose: to establish a regression safety net going forward, and to actively surface latent bugs already present in the code. Analyzed the bugs discovered through testing and fixed them.

### Interactive Brokers API

The application integrated with the Interactive Brokers API — the standard programmatic interface for the Interactive Brokers brokerage platform, commonly used in retail and institutional quantitative trading tools for market data and order management.
