## Technical Work

### EveryAngle — REST/JSON SAP Interface

The primary project was building a REST/JSON front-end layer over EveryAngle's proprietary SAP-querying interface. EveryAngle provided a unique system for querying SAP business data; the task was to wrap this in a clean, modern REST API that API consumers could interact with without needing to understand the underlying SAP integration complexity.

**Architecture design:**
Designed the architecture for this system using .NET Web API with an MVVM (Model-View-ViewModel) pattern and NHibernate for ORM/data access. This provided a structured, maintainable separation between the API layer, business logic, and data access across a team of 10–20 developers.

**Team delivery:**
Managed the team through the full development cycle, using Jira for sprint and issue tracking and Perforce for source control.

**QA:**
Directed QA using SoapUI for unit testing of the REST/JSON API endpoints, ensuring correctness of the API layer and its integration with the underlying SAP-querying system.

**Client liaison:**
Served as the primary technical liaison to EveryAngle's executive team, translating technical progress and decisions into executive-level communication and managing client expectations throughout the engagement.
